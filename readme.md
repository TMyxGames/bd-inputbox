# 自定义输入框组件

| 属性 | 说明 | 类型 | 默认值 |
| --- | --- | --- | --- |
| `title1` | 输入框内的提示文本（Placeholder） | `String` | `NULL` |
| `title2` | 输入框上边缘提示文本（聚焦、以及有内容时） | `String` | `NULL` |
| `moduleWidth` | 输入框宽度 | `String` | `10rem` |
| `moduleHeight` | 输入框高度 | `String` | `3rem` |
| `status` | 输入框状态（`default` `success` `warning` `danger`） | `String` | `default` |
| `disabled` | 禁用输入框 | `Boolean` | `false` |
| `alwaysActive` | 常态激活（使用该属性会导致`title1`无效） | `Boolean` | `false` |
| `clearable` | 启用清除内容按钮 | `Boolean` | `false` |
| `showPassword` | 启用显示密码按钮 | `Boolean` | `false` |
